UnityRSCLib

An open source RuneScape Classic library for Unity 4.x

This project aims to help Unity3D developers to create tools or clients for MoparClassic (A RuneScape Classic Server and Client)

At the moment only loading and rendering game objects (Models) are supported. Model Textures are NOT supported yet.

To use this library, import the scripts, resources and shaders into your Unity Project.

  1. Add an empty GameObject to the scene and add the Script Component: RSCModelHandler This GameObject can be placed anywhere on the scene.

  2. Add an empty GameObject to the scene and add the Script Component: RSCModelRenderer After step #2. Change the settings of the RSCModelRenderer script's "Model Name" parameter to a modelname of choice, eg. tree Note: Place this GameObject anywhere you like this model to be rendered and change the Scale if necessary.

Future updates will include

  • RSCLandscapeHandler - Section and area loading and editing. It will be used as a Terrain

  • RSCTextureHandler - Texture loading, used for both models, walls, floors and roofs.

  • RSCSpriteHandler - Sprite loading, used for billboards, players, npcs, items and ui

Moste of the code is already done, just need to adjust it to work with unity. It will come along soon enough :)
